# Опросник перед тренингом по дивергентному мышлению

Мини-сайт для GitHub Pages.

## Что внутри

- `index.html` — готовый опросник
- отдельный `app.js` не нужен: логика встроена в страницу

## Идея

Это входная диагностика перед тренингом по дивергентному мышлению.
Она смотрит не на «талант к креативу», а на то, как у участника сейчас устроен процесс работы с идеями:

- как быстро идеи разгоняются;
- насколько мешает ранняя оценка;
- насколько легко переключаться между разными ракурсами;
- насколько человек готов собирать и тестировать собственный способ генерации идей.

## Публикация

1. Открой `Settings` → `Pages`.
2. В `Build and deployment` выбери `Deploy from a branch`.
3. Укажи ветку `main` и папку `/(root)`.
4. Сохрани.

Через несколько минут GitHub Pages даст публичную ссылку.
